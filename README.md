# めっちゃX

X（旧Twitter）で画面に表示して閲覧したポストを自動保存し、Xのサイドバーから振り返れるChrome拡張です。

黒い角丸背景にポストカードと履歴時計を組み合わせた専用アイコンを収録しています。

## 主な機能

- X上に表示されたすべてのポストを、スクロール速度に関係なく閲覧履歴として保存
- 同じポストは重複せず、最終閲覧日時と閲覧回数を更新
- ポストが実際に画面内に見えていた累計時間を記録
- 遅延読み込みされる画像・動画サムネイルと複数画像（最大4枚）を記録
- キーワード、開始日・終了日、最小・最大表示秒数、今日・昨日による絞り込み
- お気に入り、ポストをXで開く、個別削除、一括削除
- 履歴記録の一時停止
- JSON形式でのバックアップ
- 最大3,000件をブラウザ内にのみ保存
- ストレージ更新をまとめ、変更された画面部分だけを検査する省負荷設計
- 拡張機能の更新で保存APIとの接続が切れた場合に自動再接続

## インストール
### chromeウェブストアからインストール
https://chromewebstore.google.com/detail/peobkadbnjniaeeicndcdcdmgfmicgpg?utm_source=item-share-cb
### 手動インストール
1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」を選ぶ
4. このフォルダーを選択する
5. Xのタブを再読み込みする

Xの左サイドバーに「ポストの履歴」が追加されます。

## プライバシー

履歴はChromeのローカルストレージに保存され、外部サーバーには送信されません。シークレットモードでは、Chromeの拡張機能設定で許可した場合のみ動作します。
